# FranzyXpress Application

This is a Vite + React web application.

## Prerequisites
- Node.js installed

## Running Locally

1. Install dependencies:
   ```bash
   npx pnpm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```
   Or if using Vite directly:
   ```bash
   npx vite
   ```

## Deploying to Vercel

This project is fully configured for Vercel.
1. Push this repository to GitHub.
2. Go to Vercel, click "Add New Project", and import your repository.
3. Vercel will automatically detect the Vite framework and the `vercel.json` configuration.
4. Click "Deploy".
